Summary of running CCE tests.

Running from Windows command promt

    1) start cmd.exe

    2) execute script build_and_run_cce_tests.bat

Running from EShell

    1) give command at your command prompt:
    \epoc32\release\winscw\udeb\eshell.exe -Dtextshell --

    2) in eshell give command ccetests

    3) Results are found from \epoc32\WINSCW\C\logs\TestFramework\testengine\testreport_all_cce_cases.html

Test reports

    Are created to \epoc32\WINSCW\C\logs\TestFramework
